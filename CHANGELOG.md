## [Unreleased]

### Added
- Initial release of UI Bug Scanner Agent Skill
- Accessibility analyzer with axe-core integration
- Usability analyzer with heuristic checks
- Custom spec analyzer with JSON DSL
- Multiple crawl modes (single, sitemap, BFS, journey)
- Multi-viewport support (desktop, tablet, mobile)
- Authentication support (cookies, headers, login scripts)
- Report generators (Markdown, JSON, SARIF)
- Finding deduplication and clustering
- Evidence capture (screenshots, DOM snippets)
- CI/CD integration examples
- Comprehensive documentation

### Features
- WCAG 2.1/2.2 Level AA compliance checking
- Tap target size validation
- Focus trap detection
- Overlapping element detection
- Modal behavior checks
- Custom UI specification validation
- GitHub Actions workflows for CI/CD
- GitHub Pages documentation site
